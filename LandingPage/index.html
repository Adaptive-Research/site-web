Le cold calling

La téléprospection pour être efficace doit reposer sur une base de données des entreprises fiable et complète. Oubliez LinkedIn. La base la plus complète est entièrement gratuite, ce sont les fichiers SIRENE. Vous pouvez la compléter avec des informations qui proviennent du bodacc ou d'infogreffe. Cela vous permet de disposer d'une base de données des entreprises qui respecte la RGPD.

Que peut faire PlusDeCA.fr pour mon entreprise ?

Pour trouver des clients, l'un des moyens les plus simples est d'avoir un site bien référencé sur les moteurs de recherche. Y parvenir nécessite souvent un fort investissement et il est facile d'être inefficace. C'est pour cela que nous avons développé une offre pour optimiser votre présence sur internet.

Un autre moyen est d'appeler ou d'envoyer un mail à des entreprises afin de leur proposer vos services. Attention toutefois à la loi RGPD ( CNIL ) qui s'applique si vous détenez les coordonnées permettant d'identifier une personne physique (ex un employé, un indépendant,...) lorsque vous prospectez. En cas de collecte indirecte (récupération par fichier, par internet,...), vous devez les informer dès votre premier appel sur le moyen que vous avez utilisé pour collecter leurs coordonnées et vous devez leur permettre d'effacer les informations que vous possédez dans votre base de données.

2) De la formation pour apprendre à écrire des articles bien classés sur Google (SEO)

Le Référencement (ou SEO pour Search Engine Optimization) est un ensemble de techniques ou de règles à respecter pour améliorer le classement d'un article dans un moteur de recherche (Google). Nous proposons de la formation au Référencement sous forme d'articles exclusifs. Pour écrire ces articles, nous avons mis sous surveillance (Veille Référencement) un certain nombre de secteurs d'activités afin d'identifier les entreprises qui ont un vrai savoir-faire. Nous analysons en détail les stratégies qu'elles emploient puis nous les testons sur des sites web que nous possédons avant d'écrire un ou plusieurs articles.
